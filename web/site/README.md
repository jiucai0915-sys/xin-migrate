# 汇报网页（深蓝官方模板风）· web/site/

> 答辩用的 10 屏汇报站点 —— 纯前端、零依赖、可断网。
> 与现场 Demo（`web/app.py` 的 streamlit 界面）分工：本页讲故事，Demo 跑真迁移。

## 怎么打开
- 双击 `index.html`，用浏览器（Chrome / Edge）打开。
- 全屏演示：打开后按 `F11`，干净无浏览器边框。
- 可断网：CSS / JS 全部内联在单文件里，不联外网。

## 操作（翻页 / 滚动两用）
| 操作 | 效果 |
|---|---|
| `空格` / `↓` / `→` / `PageDown` | 下一屏 |
| `↑` / `←` / `PageUp` | 上一屏 |
| 鼠标滚轮 | 整屏吸附滚动（scroll-snap） |
| 右侧圆点 | 点击直接跳到对应屏 |

## 10 屏结构
封面 → 痛点 → 致命一击 → 方案 → 为什么是 Agent → ★核心差异化(自修复 diff) → 两道质量关 → 技术架构 → 商业价值(ROI) → 路线图与团队。

## 视觉
深蓝官方模板风（赛事统一模板配色），与封面页一致。文案对齐 `docs/PITCH.md`、`docs/PRODUCT.md`、`docs/ROADMAP.md`。

## 改内容
直接编辑 `index.html`（单文件）：文案在各 `<section class="slide">` 里；配色在顶部 `:root` CSS 变量；队伍信息在第 1 屏。
