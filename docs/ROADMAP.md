## 接下来怎么走（P0 → P2）

### 🔴 P0 · 从"演示级"到"可用级"
1. **语义验证规模化**：当前原型覆盖单查询，扩展到自动生成测试数据 + 批量回归比对
2. **真实国产库对接**：从 SQLite/PG 模拟切到真连达梦/人大金仓执行验证
3. **覆盖范围扩展**：从 SQL 片段 → 存储过程(PL/SQL)、视图、触发器、DDL、应用层(MyBatis/ORM)

### 🟠 P1 · 规模化与可信度
4. **工程级规模化**：几十万行级批量扫描、依赖分析、断点续传、并行
5. **知识库深化**：12 条 → 数百条规则 + 版本差异库（核心护城河）
6. **模型升级**：7b → 14b/32b 或微调的迁移专用模型
7. **真实审核工作流**：人工审核队列 + 在线改写 + 协作

### 🟡 P2 · 信创合规与企业级
8. **信创部署**：离线安装包 + 国产芯片(昇腾/海光) + 国产OS(麒麟/统信)适配
9. **审计/权限/安全**：审计日志、RBAC、等保合规
10. **可回滚 + 版本管理 + 自动化测试**

### 🟢 商业
11. 信创认证（适配/安全认证）　12. 央国企标杆 POC
